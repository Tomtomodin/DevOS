Privacy Policy – Dev-OS GPT

Last updated: 2025-01-01

1. Scope

This Privacy Policy applies to the Dev-OS GPT, including all associated Actions used within the OpenAI GPT platform.

2. Nature of the Service

Dev-OS GPT is a technical assistant designed for software development, architecture analysis, code generation, validation, testing, and simulation.

3. Data Processing

The system may process the following technical inputs provided by users:

  • Source code
  • Configuration files
  • Architecture descriptions
  • CI/CD pipeline steps

This data is processed only transiently to generate responses. The service is not intended for personal data.

4. Personal Data

Dev-OS GPT does not intentionally collect, store, or process personal data. Users are advised not to submit personal or sensitive information.

5. Data Storage

No personal data is stored. Any technical input is processed ephemerally and discarded after the request is completed.

6. Data Sharing

No data is sold, shared, transferred, or disclosed to third parties by Dev-OS GPT.

7. Cookies & Tracking

Dev-OS GPT does not use cookies, tracking pixels, analytics tools, or fingerprinting mechanisms.

8. Security

Reasonable technical and organizational measures are applied to ensure the secure handling of data. However, no system can be guaranteed to be completely secure.

9. User Rights

As no personal data is stored, there is no personal data to access, rectify, or delete.

10. Changes

This Privacy Policy may be updated to reflect legal or technical changes. The most current version will always be made available at the same URL.

11. Contact

For questions regarding this Privacy Policy, please contact the operator of the Dev-OS GPT.
